# 🏋️ Multifit Equipamentos
Esse site é um catalogo on-line da empresa Multifit Equipamentos, acesse já!
 
---
 
## 📋 Sobre o Projeto
 
A Multifit Equipamentos é uma empresa brasileira especializada em equipamentos e acessórios esportivos para academias, studios, condomínios e centros de treinamento. Este site foi desenvolvido para digitalizar o catálogo de produtos e facilitar o contato para solicitação de orçamentos.
 
---
 
## ✨ Funcionalidades
 
- 🔍 **Busca em tempo real** — filtra os produtos pelo nome ou descrição instantaneamente
- 🖼️ **Banner de destaques** — carrossel automático com os produtos mais vendidos
- 🃏 **Catálogo completo** — grid de produtos com cards navegáveis
- 📄 **Página de detalhe** — especificações técnicas e características de cada produto
- 📱 **Design responsivo** — adaptado para desktop, tablet e mobile
- 📧 **Botão de orçamento** — abre o e-mail da empresa diretamente, podendo mandar um orçamento a empresa
